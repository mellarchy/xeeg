Xeeg - Template File Generator

An advanced and universal template file generator that puts you in charge

Getting Started - 5min ⏱️

Easy to Use

Zeeg was designed from the ground up to be easily configured and used to dynamically generate static files for your projects.

Focus on What Matters

Much too often, package/framework developers scare away end-user developers with their bulky syntax. By integrating Xeeg into your source, you enjoy the benefit of an awesome markup file generator which can be customised to fit your brand.

Cross Platform

Extensible to all platforms, Xeeg can be packaged and executed anywhere. Xeeg aims to remove OS and code stack barriers when it comes to template file generation.
